== Description ==

This Card flip image slideshow script utilizes CSS3 transform to rotate images with unhinge animation. It supports both auto and manual method for images display. This unhinge effect works in all browsers that support CSS3 transform. In non-supporting browsers version, a standard slide down effect is used instead.

Check official website for live demo [http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/](http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/)

* Live Demo : [Live Demo](http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/)	
* More Information : [More Information](http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/)				
* Plugin : [Plugin URI](http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/)		

1. Supports two different display modes Auto and Manual. In auto mode, slideshow pauses when the mouse rolls over the slideshow.
2. Supports persistence of last viewed slide within a browser session, so reloading the page recalls that slide.
3. Customize the unhinge animation to either drop from the left or right side or any origin.

**Plug-in features**

* Supports all major browsers.
* Responsive width.
* Slideshow auto play option.
* Widget option for sidebar.
* Short code option for posts and pages.

== Installation ==

= Install Plugin using WordPress Plugin Search =

Go to your WordPress admin area and click on Plugins >> Add New. Search Card flip image slideshow plugin using search option. Find the plugin and click Install Now link. Finally click activate plugin link to activate the plugin.

[Card flip image slideshow](http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/ "Card flip image slideshow WordPress plugin")

= Install Plugin using the WordPress Admin Plugin Upload =

Download the plugin card-flip-image-slideshow.zip from [download](http://www.gopiplus.com/work/2019/12/15/card-flip-image-slideshow-wordpress-plugin/ "card-flip-image-slideshow.zip") location. == Frequently Asked Questions ==

= Q1. How to change width and height of the slideshow? =

In the short code we have parameter to set the width and height of the slideshow. Add only number in pixel format. for mobile screen, automatically 100% width will be used.

= Q2. How to enable or disable slideshow auto play preference? =

Auto play parameter is available in the short code. pass the Boolean value true or false to enable or disable slideshow auto play preference.

= Q3. What is the use of Pause option in the short code? =

When auto play is set to true, sets the pause in milliseconds between slide changes.

= Q4. What is the use of duration parameter in the short code? =

This parameter is used for unhinge effects. Sets the duration in milliseconds of the "unhinge" effect.

= Q5. How to increase or decrease animation speed in the slideshow? =

Use the duration parameter in the short code to change the animation effect.
